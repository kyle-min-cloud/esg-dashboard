현재 계정은 시연 목적으로 단순 비밀번호를 사용합니다. 실제 운영 시에는 사내 SSO 또는 별도 인증 체계로 교체해야 합니다.
이 페이지 URL은 외부에 공유하지 마세요.

권한 비교
Admin
  • 대시보드 전체 탭
  • 취합현황 탭 조회
  • status 수정 (confirm/reject)
  • 데이터 제출 (input.html)
  • 직접 제출 시 자동 confirmed
Manager
  • 대시보드 전체 탭
  • 취합현황 탭 조회
  • status 수정 불가
  • 데이터 제출 (input.html)
  • 자동 confirmed 없음
User
  • 대시보드 기본 탭
  • 취합현황 탭 불가
  • status 수정 불가
  • 데이터 제출 (input.html)
  • 자동 confirmed 없음
